2.grayscale face image

To get a gray image of a color image, I first split the original image into three channels, blue, green and red.
These three image arrays are all gray but each of them has its own color.
For example, blue channel holds blue and red channel is a image that has more red vector.
You can either apply a average operater, which is a 3x3 matrix that that the average value and do the summation or u can simply take the average and add them up to form the grayscale image.

3.processed face image

The goal of this face processing function is to make this image flipped vertically and turn the color of it oppositely.

3.1 Reversing color

To reverse to color of a image, I take the max value of pixel values in my image and subract it with each pixel value.
Take max value as 255 makes a little difference.

3.2 Flip image vertically

I have tried two method to flip a image, one using numpy.array and the other one using cv2 only.
I think converting an image to array is eazier to calculate and operate.
The only thing is to turn each pixel symmetrically in an inverse order.
